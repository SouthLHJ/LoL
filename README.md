## 간단 소개

    Riot의 유명한 게임 Leage of Legend의 챔프들의 설명과 유저의 기본 정보를 검색할 수 있는 앱입니다. 

    간단하게 역할 군으로 챔프를 모아 놓았습니다.

    각각의 챔프를 누르면 상세 보기로 넘어가 챔프의 스탯과 스토리를 볼 수 있습니다.

## 사용 기술 및 라이브러리

- ReactNative, EXPO
- Navigation
- Riot Open API

## 담당한 기능 (Android)

- 라이엇 오픈 API를 이용하여 유저이름을 검색하면 아이디,레벨을 제공하는 기능 구현.
- 라이엇에서 제공한 챔프의 내용을 json으로 저장하여 챔프들을 역할군으로 구분.
- 챔프의 정보를 쉽게 확인하는 기능을 구현.
- 즐겨찾기 기능 구현.

## 깨달은 점

- ReactNative에서 제공하는 네비게이션을 이용하여 간단하게 경로를 구성하는 방법을 알았다.
- 라이엇 오픈 API key를 개인적으로 발급받으면 사용 기한이 매우 짧다는 것을 알았다.
- Android와 iOS 화면의 UI을 다르게 할 필요가 있다는 것을 알았다.
